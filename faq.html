What is the DOM in JavaScript?

The DOM (Document Object Model) is a programming interface for web documents.

How do you select an element by its ID in the DOM?

You can select an element by its ID using the document.getElementById('elementId') method in JavaScript.

How can you create a new HTML element using JavaScript?

You can create a new element using document.createElement('tagName'). For example, document.createElement('div') creates a new <div> element that can be appended to the DOM

How do you change the text of an element in JavaScript?

To modify the text content of an existing element, you can update its properties responsible for text, such as the inner text. This replaces the existing text content with new text, allowing you to dynamically update the page content.
